# Comments

Comments are notes that help to explain the code. Python ignores comments while running the program.

## Single-Line Comment

Use `#` to write a comment.

```python
# This is a comment
print("Hello, World!")
```

You can also write a comment at the end of a line.

```python
print("Hello, World!")  # Prints a message
```

Comments are also useful for temporarily skipping a line of code.

```python
# print("Hello, World!")
print("Python is fun!")
```

## Multi-Line Comment

Python doesn't have a special syntax for multi-line comments.

The common way is to use `#` on each line.

```python
# This is a comment.
# It continues
# on multiple lines.
```

You may also see triple quotes (`""" """`), but they're actually multi-line strings, **not real comments**.

```python
"""
This looks like a multi-line comment,
but it's actually a string.
"""
```

> **Remember:** Use `#` for comments. It's the standard and most commonly used way in Python.
