## What is Python?



Python is a programming language that's easy to learn and use. 



Created by Guido van Rossum and released in 1991. 



Its syntax is simple and easy to read, it's one of the best languages for beginners.



## What can Python be used for?



- Building websites

- Creating software and applications

- AI, Machine Learning, and Data Science

- Working with databases

- Reading and writing files

- Automating repetitive tasks (scripting)



## Why do people like Python?

- Easy to read and understand.

- Works on Windows, macOS, and Linux.

- You can write more with fewer lines of code.

- Great for both small projects and large applications.

- Widely used in the tech industry.



## Python Syntax



One thing I like about Python is that it looks clean. Instead of using lots of symbols like {} or ;, Python uses indentation (spaces) and new lines to organize code. This makes programs much easier to read and write.
