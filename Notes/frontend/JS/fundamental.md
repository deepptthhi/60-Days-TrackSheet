

## JS Home

JavaScript is the programming language of the web. It adds interactivity
to HTML and CSS.

## Introduction

-   Runs in browsers and on servers (Node.js).
-   Case-sensitive.
-   Interpreted/JIT compiled.

## Where To

JavaScript can be written: - Inline - Internal (`<script>`) - External
(`script.js`) ✅ Recommended

``` html
<script src="script.js"></script>
```

## Output

-   `console.log()`
-   `alert()`
-   `document.write()`
-   `innerHTML`

## Syntax

``` js
let name = "Deepthi";
console.log(name);
```

## Variables

-   `let` → changeable
-   `const` → cannot be reassigned
-   `var` → old keyword (avoid)

## Data Types

-   String
-   Number
-   Boolean
-   Undefined
-   Null
-   Object
-   BigInt
-   Symbol

## Comments

``` js
// Single line
/* Multi-line */
```

## Operators

Arithmetic, Assignment, Comparison, Logical, Ternary.

## Type Conversion

``` js
Number("10")
String(100)
Boolean(1)
```

## Style Guide

-   Meaningful variable names
-   Indentation
-   Semicolons (optional but consistent)
-   Prefer `const` then `let`
